# Intellij IDEA 插件开发

## 开发

根据官方文档[3]，目前插件开发支持使用Plugin Devkit，但是推荐使用Gradle方式，且搜索Plugin Devkit插件时已经找不到了，所以我选择使用gradle来开发plugin插件。
